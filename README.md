# simple-notepad

Simple notepad with the most important features.

## Installation

```bash
git clone https://github.com/V0idLib3r/simple-notepad.git 
sudo apt install python3-tk
```

## Usage
```bash
cd simple-notepad
python3 notepad.py
```
